# CV Builder

An open source CV builder for tech professionals. Paste a job description, get a tailored resume.

Built by the [Tech Immigrants](https://youtube.com/c/TechImmigrants) community.

## Why this exists

Good CV tools are expensive. Many job seekers, especially immigrants and people in countries with restricted internet access, can't use them. We're building a free, open source alternative that anyone can run.

## What it does (v1 goals)

- Paste a job description and your CV
- Get a tailored version optimized for that role
- Export as PDF, Markdown, or DOCX
- Works offline, no data leaves your browser

## Getting started

```bash
# Clone the repo
git clone https://github.com/TechImmigrants/cv-builder.git
cd cv-builder

# Install dependencies (once we have code)
# pnpm install

# Start development
# pnpm dev
```
